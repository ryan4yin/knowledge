# Emotional Text-to-Speech - 情感语音合成

随着深度学习技术的发展，不论是我们接到的骚扰电话，电子阅读软件中的语音朗读，还是地图软件的语音导航，都越来越真实了。

但是总的来讲，上述几个场景中使用的都还只是单纯的语音合成，快慢、语气等目前都能把我的比较好了，但是并没有带上情绪，更别说识别到文字所表达的情绪了。
