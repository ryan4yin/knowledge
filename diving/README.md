## 潜水类型介绍

详见 [./disciplines.md](./disciplines.md)

| 类型 | 核心特点 | 适合人群 |
|------|----------|----------|
| **自由潜** | 闭气潜水，极简装备，与海洋融为一体 | 追求精神成长、挑战自我的人 |
| **水肺潜水** | 携带气瓶，长时间水下停留，观光导向 | 喜欢探索沉船、长时间观光的人 |

**快速选择指南**：
- 追求极简、挑战自我、精神成长 → **自由潜**
- 长时间探索、观光沉船珊瑚 → **水肺潜水**
- 水下摄影 → 水肺（时间充足）或自由潜（不惊扰海洋生物）

## 课程认证体系

**为什么需要考证**：
- 国际通用认证，全球潜店认可
- 系统学习安全知识和技术
- 租赁装备、参加潜团通常需要证书
- 潜水保险保障要求

### 自由潜认证

| 等级 | 课程时长 | 目标深度 | 费用参考 | 推荐指数 |
|------|----------|----------|----------|----------|
| **AIDA 2 星** ⭐️ | 2-3天 | 16-20米 | $250-400 | ⭐⭐⭐⭐⭐ 入门首选 |
| AIDA 1 星 | 1天 | 0-10米 | $150-300 | ⭐⭐⭐ 体验级 |
| SSI Level 1 | 2-3天 | 16-20米 | $250-400 | ⭐⭐⭐⭐ 次选 |

**推荐**：直接从 **AIDA 2 星** 开始，全球认可度最高。

### 水肺潜水认证

| 等级 | 课程时长 | 深度上限 | 费用参考 | 推荐指数 |
|------|----------|----------|----------|----------|
| **PADI OW** ⭐️ | 3-4天 | 18米 | $300-500 | ⭐⭐⭐⭐⭐ 入门必选 |
| PADI AOW | 2-3天 | 30米 | $250-400 | ⭐⭐⭐⭐⭐ 进阶推荐 |
| SSI OW | 3-4天 | 18米 | $300-500 | ⭐⭐⭐⭐ 次选 |

**推荐**：先考取 **PADI OW**，再继续考取 **PADI AOW**。

## 学习路径建议

详见 [./tutorial.md](./tutorial.md)

### 自由潜入门路线

```
第1阶段：基础准备（2-4周）
├── 理论学习：潜水反射、CO₂耐受度、安全知识
├── 装备准备：面镜、呼吸管、长脚蹼、湿衣
└── 体能训练：有氧运动、核心力量

第2阶段：泳池训练（1-2月）
├── 静态闭气：30秒 → 2分钟
├── 动态闭气：0米 → 50米
└── 耳压平衡：掌握 Frenzel 技术

第3阶段：开放水域（2-4月）
├── 潜伴制度：永远不要独自潜水
├── 恒重下潜：0米 → 20米
└── 心理训练：克服深度恐惧
```

### 水肺潜水入门路线

```
PADI OW（入门）→ PADI AOW（进阶）→ 特长课程 → Rescue Diver（可选）
```

## 关键安全提醒

> ⚠️ **安全是潜水的底线，永远不要妥协**

### 自由潜安全黄金法则

| 法则 | 说明 |
|------|------|
| **潜伴制度** | 永远不要独自潜水，一人下潜，一人水面观察 |
| **1 Up 规则** | 上浮后等待 30 秒，观察无浅水 blackout（SWB） |
| **避免过度换气** | 准备呼吸不超过 2 分钟，purge 不超过 15-20 次 |
| **了解极限** | 不要为了面子或记录冒险 |

### 水肺潜水安全要点

| 要点 | 说明 |
|------|------|
| **遵守减压表** | 避免减压病（Decompression Sickness） |
| **装备检查** | 每次下潜前检查装备 |
| **潜伴制度** | 不要单独潜水 |

详见安全和技术要点：[./tutorial.md](./tutorial.md)

## 学习地点推荐

**国内**：
- 三亚：交通便利，适合入门
- 涠洲岛：珊瑚较好，性价比高
- 西沙：水质极佳，船宿，价格较高

**东南亚**（推荐，性价比高）：
- **菲律宾**：薄荷岛（自由潜胜地）、杜马盖地（价格便宜）
- **泰国**：普吉岛（设施完善）、苏梅岛（环境优美）
- **印尼**：巴厘岛（潜点多）、图蓝本（沉船潜水）

## 总结

### 快速决策表

| 你的需求 | 推荐选择 | 首选认证 |
|----------|----------|----------|
| 精神成长、极简体验 | 自由潜 | AIDA 2 星 |
| 长时间观光、探索沉船 | 水肺潜水 | PADI OW + AOW |
| 水下摄影 | 两者皆可 | - |
| 预算有限 | 自由潜（装备成本低） | AIDA 2 星 |

### 学习建议

1. **必须完成系统课程** - 安全知识必须由专业教练传授
2. **自由潜首选 AIDA 2 星** - 全球认可度最高
3. **水肺首选 PADI OW** - 覆盖面最广
4. **东南亚学习性价比最高** - 适合考证
