# 韩语元音（모음）

韩语有 21 个元音，分为**基本元音**和**复合元音**（双元音）。

## 基本元音（10 个）

这些是最基础的元音，按韩语字母表顺序排列：

| 元音 | 记忆法 |
|------|--------|
| ㅏ | 字形像竖线加一横，嘴巴张大 |
| ㅑ | 字形像「ㅏ」上面加一短横，发音较 ㅏ 更短促 |
| ㅓ | 字形像向左的「ᄀ」，嘴巴微张 |
| ㅕ | 字形像「ㅓ」上面加一短横，发音较 ㅓ 更短促 |
| ㅗ | 字形像竖线加上面一横，嘴巴圆起 |
| ㅛ | 字形像「ㅗ」上面加一短横，发音较 ㅗ 更短促 |
| ㅜ | 字形像竖线加下面一横，嘴巴收圆 |
| ㅠ | 字形像「ㅜ」上面加一短横，发音较 ㅜ 更短促 |
| ㅡ | 字形像一条短横线，嘴巴微张 |
| ㅣ | 字形像一条竖线，嘴巴扁平 |

## 元音分类

根据舌头位置和嘴巴形状，元音可以分为：

### 1. 单元音（基本元音）
- **ㅏ, ㅑ, ㅓ, ㅕ** - 前元音
- **ㅗ, ㅛ** - 后元音（圆唇）
- **ㅜ, ㅠ** - 后元音（圆唇）
- **ㅡ** - 央元音
- **ㅣ** - 前高元音

### 2. 复合元音（双元音）

由两个基本元音组合而成：

| 元音 | 组成 | 记忆法 |
|------|------|--------|
| ㅐ | ㅏ + ㅣ | 字形像「ㅏ」和「ㅣ」的结合 |
| ㅒ | ㅑ + ㅣ | 字形像「ㅑ」和「ㅣ」的结合 |
| ㅔ | ㅓ + ㅣ | 字形像「ㅓ」和「ㅣ」的结合 |
| ㅖ | ㅕ + ㅣ | 字形像「ㅕ」和「ㅣ」的结合 |
| ㅘ | ㅗ + ㅏ | 字形像「ㅗ」和「ㅏ」的结合 |
| ㅙ | ㅗ + ㅐ | 字形像「ㅗ」和「ㅐ」的结合 |
| ㅚ | ㅗ + ㅣ | 字形像「ㅗ」和「ㅣ」的结合 |
| ㅝ | ㅜ + ㅓ | 字形像「ㅜ」和「ㅓ」的结合 |
| ㅞ | ㅜ + ㅔ | 字形像「ㅜ」和「ㅔ」的结合 |
| ㅟ | ㅜ + ㅣ | 字形像「ㅜ」和「ㅣ」的结合 |
| ㅢ | ㅡ + ㅣ | 字形像「ㅡ」和「ㅣ」的结合 |

## 元音书写规则

韩语元音分为两类，书写规则不同：

### 1. 竖元音（ㅏ, ㅑ, ㅓ, ㅕ, ㅣ）
辅音写在左边：
- 가 = ㄱ + ㅏ
- 나 = ㄴ + ㅏ
- 다 = ㄷ + ㅏ
- 라 = ㄹ + ㅏ
- 마 = ㅁ + ㅏ
- 바 = ㅂ + ㅏ
- 사 = ㅅ + ㅏ
- 자 = ㅈ + ㅏ
- 카 = ㅋ + ㅏ
- 타 = ㅌ + ㅏ
- 파 = ㅍ + ㅏ
- 하 = ㅎ + ㅏ

### 2. 横元音（ㅗ, ㅛ, ㅜ, ㅠ, ㅡ）
辅音写在上边：
- 고 = ㄱ + ㅗ
- 노 = ㄴ + ㅗ
- 도 = ㄷ + ㅗ
- 로 = ㄹ + ㅗ
- 모 = ㅁ + ㅗ
- 보 = ㅂ + ㅗ
- 소 = ㅅ + ㅗ
- 조 = ㅈ + ㅗ
- 코 = ㅋ + ㅗ
- 토 = ㅌ + ㅗ
- 포 = ㅍ + ㅗ
- 호 = ㅎ + ㅗ

## 元音发音规则

### 1. ㅐ vs ㅔ
- 现代韩语中，这两个音基本相同
- 传统上 ㅐ 比 ㅔ 发音更长、更开
- 大多数情况下可以互换使用

### 2. ㅒ vs ㅖ
- 这两个音在现代韩语中使用较少
- ㅒ 主要用于「얘（孩子）」等少数词汇
- ㅖ 主要用于「예（是）」、「계（鸡）」等词汇

### 3. ㅚ vs ㅟ
- ㅚ 传统发音不同，现代韩语中发音接近
- ㅟ 发音相对稳定
- 在某些方言中两者仍然有区别

### 4. ㅢ 的发音
- 作为单词第一个音节时，发完整音
- 在单词中间时，常常简化为「ㅣ」音
- 例如：의사（医生）发完整音，但 회의（会议）中的「의」发简化音

## 学习建议

1. **先掌握 10 个基本元音**：这些是最常用的，必须熟练掌握
2. **区分竖元音和横元音**：注意书写时辅音的位置不同
3. **练习复合元音**：复合元音是基本元音的组合，理解组合规则后更容易记忆
4. **多听多说**：韩语元音的发音位置与汉语略有不同，需要多练习
5. **使用输入法练习**：在电脑上用 두벌식 输入法练习拼写，可以加深记忆（键位见输入法设置）
