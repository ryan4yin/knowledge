# 韩语辅音（자음）

韩语有 19 个辅音，分为**基本辅音**、**紧辅音**和**送气辅音**。

## 基本辅音（14 个）

这些是最基础的辅音，按韩语字母表顺序排列：

| 辅音 | 记忆法 |
|------|--------|
| ㄱ | 字形像向左撇的「7」，发音类似「哥」 |
| ㄴ | 字形像字母「L」，舌尖抵住上齿龈发鼻音 |
| ㄷ | 字形像字母「C」，舌尖轻触上齿龈 |
| ㄹ | 字形像「ㄴ」多了一横，舌尖颤动或发边音 |
| ㅁ | 字形像个方框「口」，双唇闭合发鼻音 |
| ㅂ | 字形像「ㅁ」上面出头，双唇轻闭后突然张开 |
| ㅅ | 字形像「ᄉ」，舌尖接近上齿龈发擦音 |
| ㅇ | 字形像个圆圈「○」，词首不发音，词尾发后鼻音 |
| ㅈ | 字形像「ㅅ」上面一横，舌面接近硬腭 |
| ㅊ | 字形像「ㅈ」上面出头，舌面接近硬腭并送气 |
| ㅋ | 字形像「ㄱ」多了一横，舌根送气 |
| ㅌ | 字形像「ㄷ」上面出头，舌尖送气 |
|ㅍ | 字形像「ㅂ」上面出头，双唇送气 |
| ㅎ | 字形像个圆圈加一竖，声门摩擦音 |

## 辅音分类

根据发音方式，辅音可以分为：

### 1. 喉音（舌根音）
- **ㄱ** - 软腭音
- **ㅋ** - 送气软腭音

### 2. 舌面音
- **ㅈ** - 舌面塞擦音
- **ㅊ** - 送气舌面塞擦音

### 3. 舌尖音
- **ㄴ** - 鼻音
- **ㄷ** - 舌尖塞音
- **ㅌ** - 送气舌尖塞音
- **ㄹ** - 舌尖闪音/边音

### 4. 唇音
- **ㅁ** - 双唇鼻音
- **ㅂ** - 双唇塞音
- **ㅍ** - 送气双唇塞音

### 5. 齿音
- **ㅅ** - 齿龈擦音

### 6. 喉音
- **ㅇ** - 声门音/软腭鼻音
- **ㅎ** - 声门擦音

## 紧辅音（쌍辅音，5 个）

紧辅音是在基本辅音的基础上重复书写形成的，发音时声带紧张，气流强：

| 辅音 | 对应基本辅音 | 记忆法 |
|------|--------------|--------|
| ㄲ | ㄱ | 「ㄱ」写两遍，发音更短促有力 |
| ㄸ | ㄷ | 「ㄷ」写两遍，发音更短促有力 |
| ㅃ | ㅂ | 「ㅂ」写两遍，发音更短促有力 |
| ㅆ | ㅅ | 「ㅅ」写两遍，发音更短促有力 |
| ㅉ | ㅈ | 「ㅈ」写两遍，发音更短促有力 |

## 辅音组合规则

### 1. 收音（받침）

辅音可以作为音节的结尾，称为「收音」。

**只能作为收音的辅音：**
- ㄱ, ㅋ (发音为 ㄱ)
- ㅂ, ㅍ (发音为 ㅂ)
- ㄷ, ㅌ, ㅅ, ㅈ, ㅊ, ㅎ (发音为 ㄷ)
- ㄴ, ㄹ, ㅁ, ㅇ (保持原音)

**收音示例：**
- 강 - 江（ㄱ 收音）
- 앉 - 坐（ㄴ 收音）
- 있 - 有（ㅅ 收音，发音为 ㄷ）

### 2. 双收音

有些音节末尾可以有两个辅音（复辅音）：

| 双收音 | 发音 | 示例 |
|--------|------|------|
| ㄳ | ㄱ | 몫 (mog - 份) |
| ㄵ | ㄴ | 않 (an - 不) |
| ㄶ | ㄴ | 많 (man - 多) |
| ㄺ | ㄱ | 닭 (dak - 鸡) |
| ㄻ | ㅁ | 삶 (sam - 生活) |
| ㄼ | ㄹ | 넓 (neol - 宽) |
| ㄾ | ㄹ | 읽 (il - 读) |
| ㄿ | ㅂ | 여덟 (yeodeol - 八) |
| ㅀ | ㄹ | 핥 (hal - 舔) |
| ㅄ | ㅂ | 값 (kab - 价格) |
| ㅄ | ㅂ | 없 (eop - 没有) |

## 学习建议

1. **先掌握基本辅音**：14 个基本辅音是基础，必须熟练掌握
2. **注意发音位置**：韩语辅音的发音位置与汉语略有不同，需要注意
3. **练习收音**：收音是韩语发音的难点之一，需要重点练习
4. **使用输入法练习**：在电脑上用 두벌식 输入法练习拼写，可以加深记忆（键位见输入法设置）
