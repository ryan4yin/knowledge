# 韩语

目标：短期内就是为了出国玩方便些


## 输入法

韩语的主流输入法：

- **두벌식 (Dubeolsik - 双排式)**：
  - 韩国最主流的输入方案，所有电脑跟手机默认都采用它。
  - 在手机上使用比较简单，而在电脑上盲打入门则有一定门槛。
- **로마자（Romaja - 罗马字）**：类似日语罗马音或汉语拼音，直接输入字母，输入法再自动给转换成韩语音节。
  - 适合零基础快速上手打字，但是韩国人自己几乎不用。
- 其他非主流输入法（Sebeolsik 390 / Sebeolsik Final / Sebeolsik Shin / ...）
  - 主要是优化了打字体验，譬如平衡左右手负荷、将高频词聚集在舒适打字区等。
  - 使用者极少，而且优化有限，没啥必要学。
