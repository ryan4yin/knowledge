# 日语假名对照记忆法

平假名和片假名的对照记忆，帮助你更快掌握日语假名。

## 使用说明

- **平假名**：用于表示日语固有词汇及文法助词
- **片假名**：用于书写外来语、拟声拟态词和强调

在 Linux 上使用 fcitx5-mozc 输入法时：
- 输入罗马音默认显示平假名
- 按 `Ctrl+i` 或 `F7` 转换为片假名
- 按 `Ctrl+u` 或 `F6` 转换回平假名

## 小写假名

日语中有一些小写的假名，顾名思义它们的写法与前面介绍的假名完全一致，但是要写得更小。

前面的单词示例中我们其实已经有用到部分小写假名，这里介绍下需要用小写的几种情况。

### 1. 小写的「ゃ、ゅ、ょ/ャ、ュ、ョ」
用于构成拗音（详见后面拗音章节）：
- きゃ（kya）、きゅ（kyu）、きょ（kyo）
- じゃ（ja）、じゅ（ju）、じょ（jo）
- 在 mozc 罗马音输入法中，这些小假名会自动生成，如输入 `kya` 会自动变成「きゃ」

### 2. 小写的「っ/ッ」（促音）
用于表示促音（详见下一节促音）：
- がっこう（gakkou）：学校
- サッカー（sakkaa）：soccer
- 在 mozc 罗马音输入法中，双写辅音字母会自动变成促音，如输入 `gakkou` 会自动在「か」和「こ」之间插入「っ」

### 3. 小写的「ぁ、ぃ、ぅ、ぇ、ぉ/ァ、ィ、ゥ、ェ、ォ」
用于特殊发音场合，常见于外来语：

| 小写假名 | 用途 | 示例 |
|---------|------|------|
| ぁ/ァ | fa, fi, fe, fo 的元音 | ファ（fa）、フィ（fi） |
| ぃ/ィ | wi, 的元音 | ウィ（wi） |
| ぅ/ゥ | tsu, fu 的元音变形 | ツア（tsua） |
| ぇ/ェ | she, che 的元音 | シェ（she）、チェ（che） |
| ぉ/ォ | fo 的元音 | フォ（fo） |

**Mozc 输入方法：**
- 小写的 a：输入 `xa` 或 `la`
- 小写的 i：输入 `xi` 或 `li`
- 小写的 u：输入 `xu` 或 `lu`
- 小写的 e：输入 `xe` 或 `le`
- 小写的 o：输入 `xo` 或 `lo`

**示例：**
- ファイル（fairu）：file → 输入 `fairu`
- シェア（shea）：share → 输入 `shea`
- チェック（chiekku）：check → 输入 `chiekku`
- ウィキペディア（wikipedhia）：Wikipedia → 输入 `wikipedhia`

### 4. 其他小写假名
- **ゎ**（小写 wa）：非常少见，用于特定词汇，如「花」的古读法

## 促音

促音（促音、そくおん）是日语中的语音停顿或双辅音现象，用小写的「っ」(平假名) 或「ッ」(片假名) 表示。

**发音规则：**
- 促音表示一个短暂的停顿，然后紧接着发下一个音节的首辅音
- 罗马音中用重复辅音字母表示（如：kk, ss, tt, pp, etc.）
- 发音时类似憋一口气，然后爆破出下一个音

**罗马音输入示例：**
- 「がっこう」→ 输入 `gakkou`
- 「サッカー」→ 输入 `sakkaa`
- 「ちょっと」→ 输入 `chotto`
- 「バッグ」→ 输入 `baggu`
- 「ケット」→ 输入 `ketto`

**含促音单词：**
- **かっき**（kakki）：活気（活泼）
- **カーソル**（kaasoru）：cursor (mouse cursor)
- **サッカー**（sakkaa）：soccer（足球）
- **ストップ**（sutorppu）：stop
- **スパンキング**（supankingu）：spanking
- **タッチパネル**（tacchipaneru）：touchpanel
- **バッテリー**（batterii）：battery
- **バックアップ**（bakkuappu）：backup (data)
- **コップ**（koppu）：cup（杯子）- 虽未列出，但很常见
- **ちょっと**（chotto）：a little bit（一点点）- 非常常用的口语
- **もっと**（motto）：more (已经出现在ま行单词中)

**促音记忆技巧：**
- 想象说话时突然被掐住脖子，停顿一下再继续
- 类似英语 "bookkeeper" 中间停顿的感觉
- 在书写时，促音「っ」要写得小一些，位于左下角

## 浊音

浊辅音，都是在对应的清音右上角添加两点（浊点）来表示：

- **が** / ガ（ga）、**ぎ** / ギ（gi）、**ぐ** / グ（gu）、**げ** / ゲ（ge）、**ご** / ゴ（go）
- **ざ** / ザ（za）、**じ** / ジ（ji）、**ず** / ズ（zu）、**ぜ** / ゼ（ze）、**ぞ** / ゾ（zo）
  - 注意：じ读音类似汉语「基」
- **だ** / ダ（da）、**ぢ** / ヂ（di）、**づ** / ヅ（du）、**で** / デ（de）、**ど** / ド（do）
- **ば** / バ（ba）、**び** / ビ（bi）、**ぶ** / ブ（bu）、**べ** / ベ（be）、**ぼ** / ボ（bo）

---

## 半浊音

半浊音只有は行的一组，是在辅音右上角添加小圆圈来表示的：

- **ぱ** / パ（pa）、**ぴ** / ピ（pi）、**ぷ** / プ（pu）、**ぺ** / ペ（pe）、**ぽ** / ポ（po）

---

## 拗音

拗音是通过在小的「ゃ、ゅ、ょ/ャ、ュ、ョ」前面加上「い段」的假名形成的：

- きゃ/キャ（kya）、きゅ/キュ（kyu）、きょ/キョ（kyo）
- しゃ/シャ（sha）、しゅ/シュ（shu）、しょ/ショ（sho）
- ちゃ/チャ（cha）、ちゅ/チュ（chu）、ちょ/チョ（cho）
- にゃ/ニャ（nya）、にゅ/ニュ（nyu）、にょ/ニョ（nyo）
- ひゃ/ヒャ（hya）、ひゅ/ヒュ（hyu）、ひょ/ヒョ（hyo）
- みゃ/ミャ（mya）、みゅ/ミュ（myu）、みょ/ミョ（myo）
- りゃ/リャ（rya）、りゅ/リュ（ryu）、りょ/リョ（ryo）
- ぎゃ/ギャ（gya）、ぎゅ/ギュ（gyu）、ぎょ/ギョ（gyo）
- じゃ/ジャ（ja）、じゅ/ジュ（ju）、じょ/ジョ（jo）
- びゃ/ビャ（bya）、びゅ/ビュ（byu）、びょ/ビョ（byo）
- ぴゃ/ピャ（pya）、ぴゅ/ピュ（pyu）、ぴょ/ピョ（pyo）

---

## 长音

- **平假名**：延长前面的元音（如：おかあさん、おばあさん）
- **片假名**：用长音符号「ー」表示（如：コーヒー koohii 咖啡、パーティー paatii 派对）
