## 关键注意事项

### **1. 安全永远是第一位**
- 在救生员（Lifeguard）监护下练习
- 避免过度疲劳，学习识别自身极限
- 公开水域必须使用救生浮标（Swim Buoy）

### **2. 技术优先于速度**
- 前期以分解练习（Drills）为主，如：前交叉练习（Catch-up Drill）、单臂划水（Single-arm Drill）
- 使用打腿板（Kickboard）和夹腿浮标（Pull Buoy）进行分解训练
- 每次练习包含：热身（Warm-up）→ 主项（Main Set）→ 放松（Cool-down）

### **3. 呼吸是关键**
- 避免憋气：保持持续呼气（Continuous Exhalation）
- 建立节奏：呼吸应与划水动作协调
- 双侧呼吸（Bilateral Breathing）：能显著改善身体对称性，减少单侧劳损

### **4. 装备建议**
- 泳镜（Swim Goggles）：选择合适防雾镜片
- 泳帽（Swim Cap）：保护头发，减少阻力
- 脚蹼（Fins）：初学者可用于体会打腿感觉，但不宜长期依赖

### **5. 常见错误纠正**
- 自由泳：避免"过中线"（Crossing Over），保持入水点在前方11点/1点方向
- 蛙泳：避免膝盖过度外翻，蹬腿后充分夹水（Squeeze）
- 仰泳：保持头部稳定，避免过度摆动

**总结**：科学的学习路径是蛙泳 → 自由泳 → 仰泳 → 蝶泳，每个阶段都需打好分解练习基础，切勿急于求成。
