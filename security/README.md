# 安全

作为一个 DevOps，当然也需要了解安全相关的知识。



1. 网络环境扫描
2. 代码扫描
3. Docker 镜像扫描
4. 服务器漏洞扫描


## 相关工具

- [vuls](https://github.com/future-architect/vuls): 基于 SSH 协议的漏洞扫描服务。
- [scapy](https://github.com/secdev/scapy/): Python 网络安全工具包。
- [lynis](https://github.com/CISOfy/lynis): 功能和 vuls 类似，不过是 shell 脚本写的。
